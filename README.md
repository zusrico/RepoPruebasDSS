# RepoPruebasDSS

## Resumen

Este repositorio contiene una colección de prácticas y experimentos desarrollados como parte del curso de Sistemas de Soporte a la Decisión (DSS). El proyecto se centra en la aplicación de técnicas de análisis de datos, modelado predictivo y optimización para el diseño de sistemas que faciliten la toma de decisiones informadas en contextos organizacionales y empresariales.

## Objetivos

Los objetivos principales de este proyecto incluyen:

- Explorar los fundamentos teóricos y prácticos de los Sistemas de Soporte a la Decisión.
- Desarrollar habilidades en el procesamiento y análisis de datos utilizando herramientas computacionales.
- Implementar modelos predictivos y algoritmos de optimización para resolver problemas reales.
- Diseñar interfaces de usuario que permitan la interacción efectiva con los sistemas desarrollados.
- Fomentar el aprendizaje progresivo a través de prácticas modulares, desde conceptos básicos hasta aplicaciones avanzadas.

## Metodología

El proyecto se estructura en módulos independientes, cada uno correspondiente a una práctica específica del curso. Cada módulo sigue un enfoque metodológico que incluye:

1. **Recopilación y Preparación de Datos**: Limpieza, transformación y estructuración de datasets.
2. **Análisis Exploratorio**: Identificación de patrones, tendencias y anomalías mediante estadísticas descriptivas y visualizaciones.
3. **Modelado**: Desarrollo de modelos predictivos utilizando técnicas de aprendizaje automático.
4. **Optimización**: Aplicación de algoritmos para la resolución de problemas de decisión.
5. **Evaluación y Validación**: Medición del rendimiento de los modelos y sistemas implementados.
6. **Desarrollo de Interfaces**: Creación de prototipos de dashboards y aplicaciones web para la presentación de resultados.

## Tecnologías Utilizadas

- **Lenguaje de Programación**: Python
- **Bibliotecas Principales**:
  - Pandas y NumPy para manipulación de datos
  - scikit-learn y TensorFlow para modelado predictivo
  - Matplotlib, Seaborn y Plotly para visualización
  - OR-Tools o PuLP para optimización
  - Flask o Streamlit para desarrollo de interfaces web
- **Herramientas de Desarrollo**: Jupyter Notebooks para experimentación, Git para control de versiones, VS Code como entorno de desarrollo integrado
- **Entorno de Ejecución**: Anaconda o Miniconda para gestión de dependencias y entornos virtuales

## Instalación y Configuración

Para ejecutar las prácticas incluidas en este repositorio, siga los siguientes pasos:

1. **Clonación del Repositorio**:
   ```
   git clone https://github.com/tu-usuario/RepoPruebasDSS.git
   cd RepoPruebasDSS
   ```

2. **Instalación de Dependencias**:
   ```
   pip install -r requirements.txt
   ```
   En caso de que el archivo `requirements.txt` no esté disponible, instale manualmente las bibliotecas mencionadas en la sección de Tecnologías.

3. **Configuración del Entorno**:
   - Cree un entorno virtual utilizando conda o venv.
   - Asegúrese de tener instaladas las versiones compatibles de Python (recomendado: 3.8 o superior).

## Uso

Cada práctica se encuentra en un directorio o archivo separado. Ejemplos de estructura:

- `practica1/`: Análisis exploratorio de datos.
- `practica2/`: Modelado predictivo básico.
- `practica3/`: Optimización de decisiones.

Para ejecutar una práctica, abra el correspondiente notebook de Jupyter o script de Python y siga las instrucciones incluidas. Se recomienda ejecutar los módulos en orden para una comprensión progresiva.

## Resultados Esperados

Al completar las prácticas, se espera que los estudiantes:

- Adquieran competencia en el uso de herramientas de análisis de datos para la toma de decisiones.
- Desarrollen prototipos funcionales de DSS aplicables a escenarios reales.
- Comprendan los desafíos éticos y técnicos asociados con la implementación de sistemas de soporte a la decisión.
